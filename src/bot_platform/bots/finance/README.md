# Finance Bot

Private Telegram finance bot for Indonesian daily money tracking.

## Product Capabilities

- parse expenses, income, transfers, and investment flows
- accept text, voice notes, and screenshots
- save into Google Sheets
- use AI with provider fallback
- keep pending confirmation state for ambiguous inputs
- support reply-based edit and delete
- generate today, week, month, budget, and comparison summaries

## Main Commands

- `/start`
- `/help`
- `/status`
- `/today`
- `/week`
- `/month`
- `/delete_last`
- `/delete_reply`
- `/edit_last`
- `/edit_reply`
- `/read`
- `/budget_set`
- `/budget_show`
- `/compare_month`

## Main Inputs

- `beli kopi 25000 pakai bca`
- `makan siang 45000 kemarin pakai gopay`
- voice note in Indonesian
- receipt or payment screenshot
